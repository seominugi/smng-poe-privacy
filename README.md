# 서미누기 POE 확장 개인정보 처리방침

서미누기가 제작한 Path of Exile 관련 크롬 확장 프로그램들의 개인정보 처리방침입니다.

---

## 북마크 아틀라스 (POE Bookmark Atlas)

북마크 아틀라스는 사용자의 개인정보를 수집하거나 외부로 전송하지 않습니다.

- **저장**: 북마크·검색 히스토리·UI 설정은 사용자 브라우저의 로컬 저장소(chrome.storage.local)에만 저장되며, 외부 서버로 전송되지 않습니다. 확장 프로그램을 삭제하면 저장된 데이터도 함께 제거됩니다.
- **외부 요청**: 가격 환산을 위한 환율 정보(seominugi.com)와 거래소 능력치 메타데이터(거래소 공개 API)를 요청합니다. 이 과정에서 사용자 식별 정보나 검색 내용은 전송하지 않습니다.
- **제3자 공유**: 없습니다.
- **권한 사용**: `storage`(로컬 저장), `tabs`(거래소 탭 감지), 거래소·seominugi.com 호스트(검색 데이터 캡처 및 환율 조회).

문의: 디스코드 / 네이버 카페

---

## 나랏말서미누기 POE

이 확장 프로그램은 사용자의 개인정보를 수집, 저장 또는 처리하지 않습니다.

오로지 한국어 번역 기능을 제공하고 사용자 경험을 향상시키기 위해 사용됩니다.

개인정보 처리 방침에 관한 질문이 있거나 추가 정보가 필요한 경우, 확장 프로그램 개발자에게 문의하십시오.
